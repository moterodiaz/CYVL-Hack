# Antigravity Side-Agent System - Coordination Rules

## Goal

* Claude Code agents are the primary builders.
* Antigravity side agents never redesign architecture.
* Antigravity side agents never introduce new features.
* Antigravity side agents focus on cleanliness, consistency, maintainability, readability, and project organization.
* Side agents act after implementation passes are completed.

## Coordination Rules

Claude Code Agents:

* Own implementation.
* Own architecture.
* Own debugging.
* Own feature development.
* Own testing strategy.

Antigravity Side Agents:

* Own cleanliness.
* Own organization.
* Own documentation.
* Own consistency.
* Own technical debt reporting.

## Conflict Resolution

If a side agent recommendation would alter functionality, architecture, APIs, or feature behavior:

STOP.

Create a recommendation report instead.

Claude Code remains the final authority on implementation decisions.
